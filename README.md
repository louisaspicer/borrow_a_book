##### User Stories:

```
As a user,
I would like to be able to add a book I own,
So that other users can see it
```
```
As a user
I would like to be able to see books other users own,
So that I could potentially borrow one
```
```
As a user,
I would like to be able to make a request,
To borrow another users' book
```
```
As a user,
I would like to be able to see what requests I have,
So that I can lend a book to another user
```
```
As a user,
I would like to be able to follow other users,
So that I can easily see what books they own
```
```
As a user,
I would like to see what books I have lent,
So I remember who has my books
```
```
As a user,
I would like to see what books I have borrowed,
So I don't forget to give them back
```
```
As a user,
I would like to see notifications of recently added books
So I don't miss any I might like
```
